#PreRequisition
- JDK1.6
- Maven

# How To Run
1. mvn install
2. mvn tomcat:run-war

#이클립스 사용자
이클립스를 사용하는 경우 checkout받은 root 디렉토리에서 "mvn eclipse:clean eclipse:eclipse" 를 실행해서
이클립스 클래스패스를 새롭게 생성 후 import합니다.
